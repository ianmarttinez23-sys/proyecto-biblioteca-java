# Biblioteca Java
Aplicación de consola desarrollada en Java para gestionar un pequeño catálogo de libros.

## Objetivo
Este proyecto ha sido desarrollado como práctica para consolidar los conocimientos básicos de Java antes de comenzar un CFGS de Desarrollo de Aplicaciones Multiplataforma.

## Funcionalidades
- Añadir libros
- Mostrar catálogo
- Buscar por título
- Buscar por autor
- Registrar préstamos
- Registrar devoluciones
- Mostrar estadísticas básicas

## Tecnologías
- Java
- Git
- GitHub
